Crates System

Description Addon

The Crates System Addon is a feature-rich system designed to introduce a fun and engaging loot box mechanic into Minecraft Bedrock Edition. Perfect for survival servers, mini-games, or role-playing servers, this addon adds an element of excitement as players open crates for random rewards of varying rarity.

Key Features

1. Multiple Crate Types

  • Common Crates: Basic crates with standard rewards.
  • Epic Crates: Advanced crates offering rarer and more valuable rewards.
  • Legend Crates: The rarest crates containing premium rewards highly sought after by players.

2. Dynamic Reward System

Rewards are fully customizable via the crates.js file. You can define items, adjust rarity levels, and include enchantments for an enhanced experience.

3. Simple and Interactive

Players can create and manage crates through an intuitive command system or interactive forms. Each crate offers a visually engaging prize display menu and animations during the opening process.

4. Easy to Configure

Add or remove crates by using straightforward commands, such as +addcrate or +removecrate, along with precise location assignment.

5. Balanced Loot System

Items are distributed based on configurable probability, ensuring a fair and exciting experience for all players.
